# VZOR — Система управления капиталом

## Что такое VZOR
VZOR — это платформа управления капиталом, которая охватывает три направления:

### 1. Девелопмент (Development)
Полный цикл девелоперского проекта: от идеи до продажи.
Включает 4 блока:
- **Блок 1: Инвестиционный анализ** — от идеи до решения GO/NO GO
- **Блок 2: Проектирование** — от ГПЗУ до разрешения на строительство
- **Блок 3: Строительство** — от котлована до ввода в эксплуатацию
- **Блок 4: Продажи и реализация** — от маркетинга до передачи ключей

Структура: 169 задач, 4 блока, полный цикл девелопмента.
Подробная структура описана в документе VZOR_DevProject_Structure_v2.1 и в файле `05-vzor-dev-structure.md`.

### 2. Финансы (Finance)
Управление финансовыми активами:
- Трейдинг
- Криптовалюты
- Акции
- Блокчейн-технологии

### 3. Недвижимость (Estate)
Управление активами земельного характера:
- Земельная недвижимость
- Аренда объектов
- Продажа объектов
- Квартиры, здания, сооружения

## Технологический стек
- **Frontend**: Three.js (3D-визуализация задач и зависимостей), JavaScript, HTML/CSS
- **Backend**: FastAPI (Python), REST API
- **Database**: PostgreSQL 15 (vzor_db), Redis (кэширование/сессии)
- **Infrastructure**: Docker Compose, Nginx (reverse proxy + static files)
- **AI**: Agent Zero (автономный AI-агент), DeepSeek API, Gemini API
- **Version Control**: Git

## Визуализация
Задачи проекта визуализируются в 3D-пространстве:
- Каждая задача — нода (узел) графа
- Зависимости между задачами — рёбра графа
- Размер, цвет и позиция нод отражают свойства задач (приоритет, статус, фаза)
- 3-уровневая иерархия: Фаза → Задача → Подзадача
