## Приоритеты
- **Критичная** — без этой задачи проект не может двигаться дальше
- **Высокая** — существенно влияет на качество и сроки
- **Средняя** — можно выполнить параллельно или отложить

## Ключевые изменения v2.1
1. Блок 2 перестроен по реальной хронологии: ГПЗУ → Анализ градпотенциала → Продуктовая концепция + квартирография → Эскизная концепция / АГР → Согласование АГР в Москомархитектуре (284-ПП) → Изыскания и ТУ (параллельно) → Стадия П → Экспертиза → РнС

4. Инвестанализ: добавлены подблоки «Ограничения ЗОУИТ» (1.3) и «Финмодель для банка с эскроу» (1.5)
