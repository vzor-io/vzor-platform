# VZOR — Контекст проекта

## Что такое VZOR
VZOR — платформа управления девелоперским проектом полного цикла. От инвестиционного анализа до продаж. Визуализация через 3D-граф задач на Three.js (197 задач, 358 зависимостей).

## 4 блока проекта (фазы)
1. **Инвестиционный анализ** (55 задач) — Go/No-Go, финмодель, земельный анализ, юридическая проверка, рыночный анализ
2. **Проектирование** (67 задач) — концепция, архитектура, инженерные системы, BIM, экспертиза, разрешения
3. **Строительство** (27 задач) — подготовка площадки, фундамент, каркас, фасад, инженерия, отделка, благоустройство
4. **Продажи** (20 задач) — маркетинг, брокеры, CRM, ДДУ, ипотека, заселение

## Иерархия задач
- **L0** — фазовые ноды (4 блока)
- **L1** — секции-молекулы (24 группы задач)
- **L2** — подзадачи-атомы (169 конкретных задач)

## Инфраструктура
- **Сервер:** Cloud.ru, Ubuntu 22.04, 4 vCPU, 16GB RAM, 200GB
- **Домен:** vzor-ai.com (SSL Let's Encrypt)
- **Стек:** Nginx + FastAPI + PostgreSQL + Redis + Grafana + Prometheus
- **RAGFlow:** база знаний с документами (поставщики, проектирование, расчёты, регламенты)
- **Git:** github.com/vzor-io/vzor-platform, ветка development

## Как искать информацию
- **Документы проекта** — используй инструмент ragflow_search для поиска по базам знаний RAGFlow
- **Задачи проекта** — используй инструмент vzor_tasks для работы с задачами в PostgreSQL
- **Серверные операции** — используй инструменты vzor_deploy, vzor_diagnostics, vzor_git

## Принципы работы
- Всегда отвечай на языке пользователя (русский)
- При вопросах о ценах, поставщиках, нормативах — сначала ищи в RAGFlow
- При вопросах о задачах — используй vzor_tasks
- Не придумывай данные — если не знаешь, скажи и предложи где найти
- Давай конкретные ответы с цифрами, ссылками на документы
