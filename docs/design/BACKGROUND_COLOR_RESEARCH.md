# Научное исследование: Оптимальный цвет фона для VZOR

> **Дата:** 2026-02-05
> **Цель:** Определить научно обоснованные параметры градиентного фона для долгой работы за экраном

## 2. КЛЮЧЕВЫЕ НАУЧНЫЕ ФАКТЫ

### 2.1 Почему НЕ чистый чёрный (#000000)

| Источник | Рекомендация |
|----------|--------------|
| Material Design | Использовать #121212 вместо #000000 |
| Apple HIG | Использовать системные тёмные цвета, а не абсолютный чёрный |
| WCAG | Слишком высокий контраст (21:1) вызывает напряжение глаз |
| UX Research | Чистый чёрный создаёт "дыру" на экране, нет глубины |

**Научное обоснование:**
- Контраст белого текста на чёрном фоне = 21:1 (слишком высокий)
- Оптимальный контраст для длительной работы = 4.5:1 - 7:1
- Чистый чёрный усиливает "halation" эффект (свечение вокруг белого текста)

### 2.2 Синий свет и циркадные ритмы

**Данные исследований:**
- Синий свет (450-495nm) подавляет выработку мелатонина
- Цветовая температура 5000K+ содержит избыток синего спектра
- Для вечерней работы рекомендуется 2700K-4000K

**Для VZOR:**
- Акцентный cyan (#00ffff) содержит синий компонент
- Фон НЕ должен добавлять синий — оставить его только для акцентов
- Тёплые оттенки фона компенсируют синеву акцентов

### 2.3 Визуальное утомление (PMC исследования)

**Факторы усиливающие утомление:**
1. Высокий контраст (чёрный/белый)
2. Насыщенные цвета на тёмном фоне ("вибрация")
3. Однородный плоский фон (нет глубины)
4. Резкие переходы цвета

**Факторы снижающие утомление:**
1. Умеренный контраст (тёмно-серый вместо чёрного)
2. Десатурированные (приглушённые) цвета
3. Градиенты для создания глубины
4. Плавные переходы

### 2.4 Радиальный градиент и восприятие глубины

**Принципы:**
- Светлый центр → тёмные края = естественное освещение
- Радиальный градиент создаёт "фокусную точку"
- Имитирует виньетирование (естественно для глаза)
- Добавляет ощущение пространства в 3D сцене

## 3. РЕКОМЕНДУЕМЫЕ ЦВЕТА

### 3.1 Основа: Material Design Dark Theme

```
Базовый фон:     #121212  (RGB: 18, 18, 18)
Поверхность:     #1E1E1E  (RGB: 30, 30, 30)
Карточки:        #252525  (RGB: 37, 37, 37)
```

### 3.2 Адаптация для VZOR (с учётом 3D)

**Вариант A: Нейтральный серый (минимум оттенка)**
```
Центр градиента: #0D0D0D  (RGB: 13, 13, 13)
Край градиента:  #000000  (RGB: 0, 0, 0)
```

**Вариант B: Тёплый тон (компенсация синих акцентов)**
```
Центр градиента: #0F0D0B  (RGB: 15, 13, 11)  — лёгкий тёплый оттенок
Край градиента:  #050403  (RGB: 5, 4, 3)
```

**Вариант C: Холодный тон (поддержка sci-fi эстетики)**
```
Центр градиента: #0A0C0F  (RGB: 10, 12, 15)  — минимальный синий
Край градиента:  #020304  (RGB: 2, 3, 4)
```

### 3.3 РЕКОМЕНДАЦИЯ ДЛЯ VZOR

**Выбор: Вариант B (тёплый тон)** с минимальной интенсивностью

**Обоснование:**
1. VZOR уже имеет cyan (#00ffff) акценты — нужен баланс
2. Тёплый фон снижает общую "холодность" интерфейса
3. Минимальный оттенок не отвлекает, но добавляет глубину
4. Компенсирует синий свет от акцентов

## 4. ТЕХНИЧЕСКИЕ ПАРАМЕТРЫ

### 4.1 Градиент для Three.js

```javascript
// Рекомендуемые значения (Вариант B: тёплый)
uColor1: new THREE.Color(0x0F0D0B)  // Центр: тёплый тёмно-серый
uColor2: new THREE.Color(0x020201)  // Край: почти чёрный с теплом

// Альтернатива (Вариант A: нейтральный)
uColor1: new THREE.Color(0x0D0D0D)  // Центр: нейтральный серый
uColor2: new THREE.Color(0x000000)  // Край: чёрный
```

### 4.2 Параметры градиента

```
Тип: Радиальный (ellipse)
Центр: 50% 50% (центр экрана)
Распределение: smoothstep с множителем 1.2-1.5
Виньетирование: мягкое (не резкий переход)
```

### 4.3 Соотношение размеров

```
Видимая зона светлого центра: ~40-50% viewport
Переходная зона: ~30-40% viewport
Тёмный край: ~10-20% viewport
```

## 5. КОНТРАСТ И ДОСТУПНОСТЬ

### 5.1 Проверка контраста

| Элемент | Цвет | Фон | Контраст | WCAG |
|---------|------|-----|----------|------|
| Основной текст | #FFFFFF | #0F0D0B | 19.2:1 | AAA ✓ |
| Приглушённый текст | #888888 | #0F0D0B | 6.1:1 | AA ✓ |
| Cyan акцент | #00FFFF | #0F0D0B | 16.5:1 | AAA ✓ |
| Task points | #4ECDC4 | #0F0D0B | 10.8:1 | AAA ✓ |

### 5.2 Снижение контраста для комфорта

Рекомендуется уменьшить яркость основного текста:
```
Вместо #FFFFFF (чистый белый)
Использовать #E0E0E0 (приглушённый белый)
Контраст: 15.1:1 — всё ещё AAA, но мягче
```

## 6. СРАВНЕНИЕ С ИСХОДНЫМ СОСТОЯНИЕМ

| Параметр | Было (#000000) | Станет (градиент) |
|----------|----------------|-------------------|
| Контраст с белым | 21:1 (чрезмерный) | 15-19:1 (оптимальный) |
| Глубина | Плоский | Объёмный |
| Утомляемость | Высокая | Пониженная |
| Эстетика | "Дыра" | Пространство |
| Цветовая температура | Нейтральная | Слегка тёплая |

## 7. ИТОГОВЫЕ РЕКОМЕНДАЦИИ

### Для v3.20:

```javascript
// Three.js Shader - ФИНАЛЬНЫЕ ЗНАЧЕНИЯ
const gradientMaterial = new THREE.ShaderMaterial({
    uniforms: {
        uColor1: { value: new THREE.Color(0x0F0D0B) },  // Центр
        uColor2: { value: new THREE.Color(0x020201) },  // Край
        uRatio: { value: window.innerWidth / window.innerHeight }
    },
    fragmentShader: `
        varying vec2 vUv;
        uniform vec3 uColor1;
        uniform vec3 uColor2;
        uniform float uRatio;
        void main() {
            vec2 uv = (vUv - 0.5) * vec2(uRatio, 1.0);
            float dist = length(uv) * 1.3;  // Множитель для мягкости
            vec3 color = mix(uColor1, uColor2, smoothstep(0.0, 1.0, dist));
            gl_FragColor = vec4(color, 1.0);
        }
    `
});
```

### Ожидаемый результат:
- Незаметный, но присутствующий градиент
- Центр экрана слегка светлее (фокус на облаке)
- Края уходят в темноту (естественное виньетирование)
- Тёплый оттенок компенсирует cyan акценты
- Снижение визуального утомления при длительной работе
