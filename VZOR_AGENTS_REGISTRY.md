# VZOR — Полный реестр субагентов
**Обновлено:** 2026-02-17
**Статус:** ЧЕРНОВИК — требует корректировки

---

## Архитектура: 3-уровневая иерархия

```
agent0 (VZOR AI) — видит 4 блок-менеджера
    │
    ├── invest-gip     → 9 специалистов по инвестанализу
    ├── design-gip     → 18 проектировщиков
    ├── expert-gip     → 8 специалистов по согласованиям
    ├── build-dir      → 17 направлений стройки
    └── sales-dir      → 10 направлений продаж
```

**Итого: 5 менеджеров + ~62 специалиста = ~67 субагентов**

---

## Модель: DeepSeek Chat, контекст 100K токенов

- Промпт субагента (role.md): 2-4 KB (~1K токенов)
- Контекст работы: ~70K токенов (достаточно для работы с документами)
- Каждый субагент получает свежий контекст при вызове
- База знаний: через ragflow_search — ТОЛЬКО свою KB
